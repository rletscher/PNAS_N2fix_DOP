# PNAS_N2fix_DOP

This repository holds the source code, necessary model input files, and model output files for the inverse model of the ocean phosphorus cycle that assimilates marine N2-fixation rate measurements to diagnose marine N2-fixation rates and patterns including that which is supported by DOP uptake vs DIP uptake. 

Four models are included which differ by their representation of the iron limitation term and/or a dependency for diazotroph DIP uptake on the distribution of the tracer P*. For each of the four models, nine unique solutions are included and can be generated by altering the line of code that reads:

      obs_N2.Jfix = obs_N2.Jfix ./ 20.0 # convert to P units with Diaz N:P = 20:1

to a value of 30.0, 40.0, etc. up to 100.0 in increments of 10, which are the prescribed diazotroph biomass N:P stoichiometry.

Model 1 = Iron limitation term is a normalized function of the euphotic zone dissolved [Fe] distribution as simulated by the inverse model-data product of Pasquier, B., & Holzer, M. (2017). Inverse-model estimates of the ocean's coupled phosphorus, silicon, and iron cycles. Biogeosciences, 14(18), 4125-4159.

Model 2 = Iron limitation term is a normalized function of the supply Fe:NO3 stoichiometry computed using the upper 108 meter stocks of [dFe] (as above) and World Ocean Atlas annual climatology for [NO3]. 

Model 3 = Analagous to Model 1 but adds a dependency on diazotroph DIP uptake to that of the P* tracer within the euphotic zone, restricting diazotroph DIP uptake to where P* is positive and scaling diazotroph DIP uptake higher at higher P*. 

Model 4 = Analagous to Model 2 with the P* dependency included.
